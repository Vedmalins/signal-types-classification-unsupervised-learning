# signal-types-classification-unsupervised-learning
Кластеризация сцинтилляционных сигналов методами машинного обучения без учителя для разделения гамма-квантов, нейтронов и аномальных событий.


# Кластеризация сцинтилляционных сигналов методами обучения без учителя

## О проекте

Проект выполнен в рамках дисциплины «Машинное обучение без учителя».

Цель исследования — автоматическая кластеризация сигналов сцинтилляционного детектора на три группы:

* сигналы первого типа частиц;
* сигналы второго типа частиц;
* аномальные или неидентифицируемые сигналы.

В качестве исходных данных используется набор сигналов, полученных со сцинтилляционного детектора на основе органических кристаллов паратерфенила. Каждый объект представляет собой зарегистрированный сигнал, описанный набором числовых признаков и временных отсчётов формы импульса.

---

## Постановка задачи

Необходимо разделить 23 479 сигналов на три кластера:

* **Кластер 0** — сигналы первого типа частиц;
* **Кластер 1** — сигналы второго типа частиц;
* **Кластер 2** — аномальные или смешанные сигналы.

Задача решается методами машинного обучения без учителя с последующей интерпретацией результатов кластеризации.

---

## Используемые методы

В рамках проекта планируется выполнить:

### Разведочный анализ данных (EDA)

* анализ структуры данных;
* поиск пропусков и дубликатов;
* статистический анализ признаков;
* визуализация формы сигналов;
* корреляционный анализ.

### Предобработка данных

* очистка данных;
* масштабирование признаков;
* анализ выбросов;
* подготовка данных к кластеризации.

### Feature Engineering

Извлечение физических характеристик сигнала:

* максимальная амплитуда;
* площадь под импульсом;
* энергия сигнала;
* длительность импульса;
* статистические характеристики формы сигнала.

### Снижение размерности

* PCA;
* t-SNE;
* UMAP.

### Алгоритмы кластеризации

* K-Means;
* Gaussian Mixture Model;
* Agglomerative Clustering;
* DBSCAN;
* HDBSCAN.

### Оценка качества кластеризации

* Silhouette Score;
* Davies-Bouldin Index;
* Calinski-Harabasz Index;
* визуальный анализ кластерной структуры.
