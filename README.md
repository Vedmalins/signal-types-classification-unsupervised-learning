# Signal Types Classification — Unsupervised Learning

Кластеризация сцинтилляционных сигналов методами машинного обучения без учителя для разделения различных типов физических событий.

## О проекте

Проект выполнен в рамках дисциплины **«Машинное обучение без учителя»**.

Цель работы — исследовать структуру сигналов сцинтилляционного детектора и разделить зарегистрированные события на три группы с помощью методов кластеризации:

* кластер 0 — сигналы первого типа;
* кластер 1 — сигналы второго типа;
* кластер 2 — аномальные или неопределённые сигналы.

Исходные данные предоставлены в рамках соревнования Kaggle **Signal Types Classification** и содержат 23 479 зарегистрированных сигналов сцинтилляционного детектора.

---

## Постановка задачи

Необходимо решить задачу кластеризации без использования размеченных данных.

Для этого были выполнены:

* исследовательский анализ данных (EDA);
* предобработка сигналов;
* извлечение инженерных признаков;
* снижение размерности методом PCA;
* сравнение нескольких алгоритмов кластеризации;
* подбор гиперпараметров;
* формирование итогового решения для Kaggle.

---

## Выполненные этапы исследования

### 1. Разведочный анализ данных (EDA)

Проведён анализ структуры датасета:

* исследована размерность данных;
* выполнена проверка пропусков и дубликатов;
* проанализирована форма сигналов;
* исследованы статистические характеристики признаков;
* построены корреляционные матрицы.

### 2. Предобработка данных

Выполнены:

* удаление неинформативных признаков;
* масштабирование данных с использованием RobustScaler;
* подготовка признаков к кластеризации.

### 3. Feature Engineering

Из формы импульсов были извлечены физически интерпретируемые признаки:

* базовый уровень сигнала (baseline);
* максимальная амплитуда импульса;
* положение максимума;
* площадь под импульсом;
* энергия сигнала;
* длительность импульса;
* время восстановления;
* статистические характеристики формы сигнала.

### 4. Снижение размерности

Для анализа структуры данных использован метод главных компонент (PCA).

Первые две компоненты объясняют более 87% дисперсии инженерных признаков.

### 5. Кластеризация

Были исследованы следующие алгоритмы:

* K-Means;
* Gaussian Mixture Model (GMM);
* Agglomerative Clustering;
* DBSCAN.

Для оценки качества использовались:

* Silhouette Score;
* Davies–Bouldin Index;
* Calinski–Harabasz Index.

## Основные результаты

Лучшие результаты среди протестированных моделей показали:

| Модель                   | Silhouette Score |
| ------------------------ | ---------------- |
| K-Means (k=3)            | 0.569            |
| GMM (tied covariance)    | 0.562            |
| Agglomerative Clustering | 0.453            |

В качестве итоговой модели была выбрана **K-Means**, показавшая лучший баланс качества кластеризации и интерпретируемости результатов.
